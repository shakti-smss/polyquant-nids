A novel hybrid Deep Learning framework combining Deep & Cross Networks (DCN-V2) and CatBoost for balanced Network Intrusion Detection.

DOI PythonTensorFlow

📄 Paper
"PolyQuant: A Hybrid Deep Cross Network and Gradient Boosting Approach for Balanced Network Intrusion Detection in IoT Environments"

Accuracy: 82.59% (CICIoT2023)
Published: [Link to Journal/PDF]
🌟 Key Features
Quantile Transformer: Robust normalization for non-Gaussian network traffic.
Polynomial Interactions: Explicitly models feature interactions (e.g., Port × Packet Size).
Class Imbalance Handling: Achieves 94%+ precision on rare attacks (SQLi, XSS).
